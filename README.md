# APP描述

APP简介口号：开启一场不一样的旅行...

APP作用： 

1. 帮助用户在旅行前制作攻略，规划路线，消费，交通等
2. 提供旅行社交功能，用户可以查找周围正在使用该APP的用户，具有添加好友，邀请加入旅行等功能
3. 可以对其它用户发起旅行邀请，添加旅行伙伴，加入旅程的用户也可以编辑该旅行攻略，即一起协作制作攻略。
